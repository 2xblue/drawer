## Download

There are a couple ways to download drawer.

- [download][1] the ZIP released.
- clone the reop `https://github.com/blivesta/drawer.git`

  ``` bash
  $ npm install
  $ grunt build
  $ grunt
  ```

And finally, open `http://0.0.0.0:9999` in your browser.
