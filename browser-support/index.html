Drawer - Browser support

A small CSS3 and jQuery plugin for app style drawer menu.

Open

Compatible Browsers

Drawer is designed to support the cross-browser. Compatible with IE7+ including old mobile browsers. Global Web Browser stats by Browser market share.

  • Chrome
  • Firefox
  • Safari
  • Opera
  • IE
  • mode: navbar IE7+
  • mode: responsive IE7+
  • mode: default IE9+

  <a class="sr-only sr-only-focusable" href="#content">Skip to main content</a>

  <header>
    <nav>
      navigation
    </nav>
  </header>

  <main id="content">
    content
  </main>
</body>

2nd tab + Enter key

toggle navigation

<div class="drawer-header">
  <button type="button" class="drawer-toggle drawer-hamberger">
    <span class="sr-only">toggle navigation</span> 
    <span class="hamberger"></span>
  </button>
</div>
