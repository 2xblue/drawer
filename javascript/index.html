Drawer - Javascript

A small CSS3 and jQuery plugin for app style drawer menu.

Open

Call the Drawer

$(document).ready(function() {
  $(".drawer").drawer();
});

API Methods

// Option
$(".drawer").drawer({
  apiToggleClass: "element"
});

// Open
$('.element').on(function)() {
  $('.drawer').drawer('open');
});

// close
$('.element').on(function)() {
  $('.drawer').drawer('close');
});

// toggle
$('.element').on(function)() {
  $('.drawer').drawer('toggle');
});

// destroy
$('.element').on(function)() {
  $('.drawer').drawer('destroy');
});

Events

// Opened
$('#element').on('drawer.opened');

// Closed
$('#element').on('drawer.closed');

Configuring the iScroll

Do not use click: true, preventDefault: true options in order to improve cross browser and web accessibility. If you want to use iscroll lite version, can build with Wheel. It's a lightweight custom iscroll.

drawer.js

var drawerScroll = new IScroll("."+options.mastaClass, {
  mouseWheel:true,
  preventDefault: false
});
